# Student Success Copilot
**Explainable Hybrid AI System — Plans, Predicts, and Explains**

---

## Setup (Mac + VS Code)

### 1. Clone / open the project
Open the `student_success_copilot/` folder in VS Code.

### 2. Create a virtual environment
Open the VS Code terminal (`Ctrl+`` ` or **Terminal → New Terminal**) and run:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

You should see `(.venv)` in your terminal prompt.

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Generate the dataset
```bash
python -m ml.dataset
```
This creates `data/student_data.csv` (1000 rows).

### 5. Run the full pipeline (demo scenarios)
```bash
python main.py
```
Runs both Demo A (low-risk student) and Demo B (at-risk student) end-to-end.
No input required — output is printed to the terminal.

### 6. Run the interactive app
```bash
python app.py
```
Menu-driven CLI — choose a demo scenario or enter your own student profile.

## AI components

| Component | Module | Algorithm |
|---|---|---|
| Search planner | `planner/planner.py` | BFS vs A* — compared on time, states, conflicts |
| Expert system | `rules/engine.py` | Forward chaining + backward chaining |
| ML classifier | `ml/model.py` | Decision Tree, Logistic Regression, Random Forest |
| GenAI explainer | `ui/display.py` | Claude API with guardrails (Option 3) |
| NLP interface | `ui/input_layer.py` | Constraint extraction + follow-up loop (Option 2) |
